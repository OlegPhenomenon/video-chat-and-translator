# Brief: Загрузка видео и доступ к ранее загруженным видео

## Проблема

В приложении отсутствует точка входа в ключевой пользовательский сценарий: авторизованный пользователь не может загрузить ни одного видео для дальнейшей работы с сервисом. 100% авторизованных пользователей не имеют интерфейса для загрузки видеофайла. Пока этот сценарий отсутствует, продукт не даёт пользователю начать основную работу с видео.

## Для кого

Авторизованные пользователи, которые хотят загрузить своё видео в приложение как первый шаг перед дальнейшей работой с ним.

## Контекст

В проекте уже есть базовая инфраструктура: аутентификация, профиль, дашборд и навигация. Однако основной сценарий продукта, связанный с пользовательским видео, ещё не представлен в интерфейсе. Это важно сейчас, потому что без возможности загрузить видео пользовательский контур остаётся подготовительным.

## Желаемый результат

Авторизованный пользователь может загрузить видеофайл, после чего загруженное видео остаётся доступным в том же браузере на том же устройстве после перезагрузки страницы. Пользователь видит все ранее загруженные видео и может открыть любое из них для воспроизведения.
