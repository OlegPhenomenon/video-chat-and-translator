# Brief: Система сброса пароля

## Проблема

Зарегистрированный пользователь, забывший пароль, полностью заблокирован — нет возможности восстановить доступ к аккаунту. Это критично: пользователь теряет все свои данные и настройки.

## Для кого

Для зарегистрированных пользователей с подтверждённым email, которые забыли пароль и не могут войти в систему.

## Контекст

Система авторизации уже реализована через Devise с подтверждением email. Пользователи начали регистрироваться. Отсутствие сброса пароля — блокер для нормального пользования сервисом: при утере пароля единственный путь — создать новый аккаунт.

## Желаемый результат

Зарегистрированный пользователь с подтверждённым email может самостоятельно сбросить пароль через email-ссылку. Для неподтверждённых пользователей сброс пароля недоступен.
