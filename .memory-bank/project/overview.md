# Описание проекта

Проект представляет собой инновационную клиент-ориентированную (client-side) образовательную платформу и сервис видеоперевода. Система позволяет преодолеть языковой барьер, предоставляя пользователям качественный дубляж контента на родном языке. Главная архитектурная особенность продукта — отсутствие тяжелого бэкенда. Все ресурсоемкие ИИ-операции (транскрибация, перевод, синтез речи) выполняются исключительно локально на мощности устройств пользователей с помощью моделей ONNX WASM прямо в браузере, либо через подключение их собственных API-ключей. Серверная часть используется по минимуму и отвечает только за хранение базовой информации о прогрессе и аккаунтах.

Платформа поддерживает максимально гибкий импорт контента: прямую загрузку файлов, перехват видеороликов через специальное браузерное расширение, а также "умную" интеграцию с YouTube. В случае с YouTube видео не скачивается на сервер — сервис использует стандартный iframe, налету парсит субтитры и синхронно накладывает сгенерированную аудиодорожку дубляжа поверх оригинального плеера. Во время просмотра с пользователем работает встроенный ИИ-ассистент, выполняющий роль персонального тьютора. Он умеет отвечать на вопросы по видео, объяснять сложные моменты и вести умные конспекты.

Для глубокого объяснения тем ИИ-тьютор имеет доступ к выделенной интерактивной песочнице (с поддержкой как JS/React, так и Python-библиотек вроде Manim). Там нейросеть может писать и исполнять код "на лету" для рендеринга наглядных графиков, анимаций и мини-презентаций с собственным голосовым сопровождением. Образовательная экосистема также нацелена на долгосрочное усвоение материала: опираясь на бэкенд со статистикой, сопутствующий Telegram-бот периодически присылает пользователю проверочные вопросы по пройденным видео (система интервальных повторений Anki). В архитектуру также заложен потенциал для будущих интеграций (Model Context Protocol), что в перспективе позволит напрямую экспортировать конспекты платформы в личные базы знаний вроде Obsidian или Notion.
