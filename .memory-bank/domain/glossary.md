# Glossary (Единый словарь терминов)

Данный документ фиксирует Ubiquitous Language (Единый язык) проекта. Все обсуждения, PRD, архитектурные документы, названия переменных и структура базы данных должны опираться на эти определения.

## Архитектура и Инфраструктура

- **Inertia (Inertia.js)** — Технология связывания (bridge) бэкенда и фронтенда. Позволяет Rails-монолиту отдавать данные как свойства (props) напрямую в React-компоненты. Заменяет собой традиционные REST и GraphQL API.
- **Client-side ML (ONNX WASM)** — Концепция локального машинного обучения платформы. Нейросетевые веса компилируются в WebAssembly (WASM) и запускаются в Web Workers или на WebGPU прямо в браузере пользователя, минуя серверы платформы.
- **Fallback UX** — Спасательный сценарий интерфейса, срабатывающий при падении тяжелых моделей (WASM OOM) на слабом устройстве пользователя (вывод уведомления или запрос стороннего API-ключа).

## Медиа и Аудио

- **Audio Ducking** — Динамическое снижение (приглушение) громкости исходного аудиопотока оригинального видео во время наложения новой сгенерированной голосовой дорожки перевода.
- **Smart iframe (Overlay Pattern)** — Метод обхода ограничений (Terms of Service) хостингов вроде YouTube. Видео не скачивается сервером, а воспроизводится в штатном iframe пользователя, поверх которого невидимым слоем ложится интерфейс платформы и WebAudio.
- **Browser Extension Hack (Content Interception)** — Механика браузерного расширения, которая инжектит React-компоненты плеера прямо в исходный (DOM) код сторонних обучающих сайтов на лету.

## Обучение и Продукт

- **AI Tutor Sandbox** — Изолированная «песочница» (WebContainers/Pyodide), в которой встроенный ИИ-тьютор может «в прямом эфире» исполнять JavaScript или Python код, чтобы динамически рисовать интерактивные графики и визуализации для учащегося.
- **Spaced Repetition (Интервальные повторения / Anki)** — Методика запоминания слов и формул. В контексте платформы реализуется через интеграцию с Telegram: бэкенд на Rails строит карточки по субтитрам просмотренных видео и рассылает их через Telegram-бота по графику забывания.
