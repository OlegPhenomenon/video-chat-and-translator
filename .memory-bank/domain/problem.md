# Project Problem Statement

Этот документ фиксирует общий продуктовый контекст платформы видеоперевода и образовательного ассистента.

## Boundary With PRD

- `domain/problem.md` — общий для всего проекта контекст: продукт, ключевые workflows, top-level outcomes и устойчивые ограничения.
- `prd/PRD-XXX-short-name.md` — инициативный слой: какая именно продуктовая проблема берется в работу сейчас, для каких пользователей и с каким scope.

## Product Context

Языковой барьер затрудняет усвоение зарубежного образовательного видеоконтента. Существующие решения часто дороги, требуют мощных серверов, не заботятся о приватности и не обладают интерактивными возможностями для обучения.

Наша система представляет собой инновационную клиент-ориентированную (client-side) образовательную платформу и сервис видеоперевода. Система выполняет все ресурсоемкие ИИ-операции (транскрибация, перевод, синтез речи) локально на устройствах пользователей с помощью моделей ONNX WASM прямо в браузере или через их личные API-ключи. Это обеспечивает приватность и радикально снижает затраты на бэкенд, который выполняет лишь легковесные функции управления аккаунтами и прогрессом. В процесс встроен ИИ-тьютор с интерактивной песочницей, помогающий пользователю активно усваивать материал прямо во время просмотра.

## Core Workflows

- `WF-01` **Smart YouTube Integration**: Просмотр обучающих видео на YouTube внутри платформы. Видео не скачивается, используется стандартный iframe, на лету парсятся субтитры и накладывается аудиодорожка локально сгенерированного дубляжа.
- `WF-02` **Active Learning with AI Tutor**: Взаимодействие со встроенным ИИ-тьютором во время просмотра. Тьютор может писать и исполнять код (JS/React, Python) "на лету" в выделенной песочнице для визуализации графиков, математики и анимаций.
- `WF-03` **Content Import**: Перехват видеороликов через браузерное расширение или прямая загрузка локальных файлов для дубляжа.
- `WF-04` **Spaced Repetition**: Прохождение проверочных вопросов от Telegram-бота, который опирается на статистику пройденных видео (Anki).

## Constraints

- `PCON-01` **Client-side computing constraint**: Строгое ограничение на выполнение тяжелых ML моделей на бэкенде. Все вычисления (Whisper, TTS и т.д.) только локальные или через API ключи пользователя.
- `PCON-02` **YouTube ToS constraints**: Запрет на скачивание YouTube потока видео и аудио файлов. Допускается только использование официального iframe плеера и on-the-fly перекрытие звука.
- `PCON-03` **Local Storage**: Ограничения браузера на хранение больших кэшей аудио/моделей (IndexedDB / Cache API).
