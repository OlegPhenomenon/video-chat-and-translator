# Brief: Профиль пользователя

**Related Issue:** [GitHub Issue #3](https://github.com/OlegPhenomenon/video-chat-and-translator/issues/3)
**Status:** active
**Date:** 04.04.2026

### Цель
Функциональность управления учётными данными отсутствует полностью — 100% пользователей не могут самостоятельно сменить email или пароль без прямого вмешательства в БД. Это делает невозможным смену скомпрометированного пароля, исправление неверного email и восстановление доступа к аккаунту после смены почты.

### Для кого
Для зарегистрированных пользователей, которые пользуются нашим сервисом для перевода и озвучки видео, а также для общения с ИИ-чатом о контенте данного видео.

### Технический контекст
Система авторизации реализована с использованием гема Devise (фича [001-auth-email](../001-auth-email/brief.md)). Задача стала приоритетной, так как пользователи начали регистрироваться, и отсутствие возможности изменить учётные данные является блокером для нормального пользования сервисом.

### Желаемый результат
Пользователь с подтверждённой почтой может самостоятельно обновлять свои учётные данные: **email** (просматривать текущий и изменять) и **пароль** (изменять, не просматривая его значение) — без обращения в поддержку.
