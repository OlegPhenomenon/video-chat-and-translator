# Brief: Плейсхолдер после регистрации

**Related Issue:** [GitHub Issue #15](https://github.com/OlegPhenomenon/video-chat-and-translator/issues/15)
**Status:** draft
**Date:** 14.04.2026

### Проблема
После успешной регистрации единственное подтверждение находится во временном toast, а основная область страницы возвращается к пустой форме. Если пользователь не заметил toast, на странице нет постоянного подтверждения успешной регистрации и следующего шага: проверить почту.

### Для кого
Для новых пользователей, которые регистрируются в сервисе.

### Контекст
Задача заведена по GitHub Issue #15 как исправление UX-дефекта в onboarding: новый пользователь после регистрации может остаться без постоянного подтверждения успешного действия и next step. Регистрация и подтверждение email уже реализованы, но текущая обратная связь зависит исключительно от toast-уведомления, которое исчезает через несколько секунд.

### Желаемый результат
После успешной отправки регистрации в основной области страницы должно отображаться постоянное сообщение, которое явно сообщает:
1. Регистрация прошла успешно.
2. Для завершения регистрации нужно проверить почту и подтвердить email.

Это сообщение должно оставаться видимым после исчезновения toast-уведомления и не должно зависеть от временного уведомления. Визуальное оформление должно соответствовать существующему стилю страницы регистрации.
